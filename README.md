# Single-Cycle-RISC-V-32I-CPU
Single Cycle RISC-V 32I CPU는 RISC-V 명령어 집합 구조(ISA) 중 가장 기본이 되는 32비트 정수형(Integer) 명령어 셋을 처리하는 프로세서로, 하나의 명령어를 처리하는 데 정확히 1 클럭 사이클이 소요되는 구조
